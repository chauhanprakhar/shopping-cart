# shopping-cart
A simple cart based web application for assessment.
 ```shell
git clone https://github.com/chauhanprakhar/shopping-cart.git
```
## Frontend
Commands
 ```shell
 cd frontend
 npm install
 npm start
```
### Tools Used
- [ Redux Toolkit](https://redux-toolkit.js.org/)
- [ React Redux](https://react-redux.js.org/)

## Backend
Commands
 ```shell
 cd backend
 npm install
 nodemon index.js
```
